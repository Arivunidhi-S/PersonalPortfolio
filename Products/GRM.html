Global Resources Management

Global Resources Management is a complete solution to track the applications of employees who have joined, resigned, been terminated, or are about to join your company. A web-based application that allows you to manage your company human resources quickly and easily. This system helps track the progress of an employeeâ€™s software systems to keep your employee file up to date. Our application allows you to enter the relevant information on your employees. Businesses are looking to increase workforce efficiency. Scheduling conflicts and mismatched manpower can cause job site inefficiency, which increases cost and negatively impacts deadlines. Our Global Resources application allows your management team to ensure deadlines match manpower needs.

Modules

  • Login Module: Secure authentication system for Admin, HR, and Staff users.
  • User Module: Manage different user roles and permissions.
  • Staff Personal Module: Store and update personal details like contact, address, DOB, etc.
  • Staff Official Module: Maintain official data like joining date, department, role, and status.
  • Company Module: Manage company-level details and configuration.
  • Project Module: Track current projects, assign staff, and manage deadlines.

Responsibilities

  • Developed modular components using ASP.NET Web Forms, with separation of concerns
  • Created login authentication and role-based access control in the Login Module
  • Implemented CRUD operations for Staff Personal and Staff Official modules using ADO.NET
  • Built user management interfaces for admins to assign roles and privileges (User Module)
  • Designed SQL Server schemas to manage staff, company, and project data with relational integrity
  • Integrated resume upload and preview within the Staff Personal module
  • Developed the Project Module to assign staff to projects and track manpower alignment
  • Implemented company-wide configuration through the Company Module
  • Applied form validations, session management, and security best practices
  • Coordinated with the HR team to ensure the system matched real workflow needs
